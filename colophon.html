This page is built using Jekyll - see the full source code under obrhubr/obrhubr.github.io. All content is fully accessible without Javascript, which is only used to display image's alt-text.

I provide an RSS feed for the site. You can follow for updates, using your feed reader of choice.

Design

The design of the site is inspired by Michael Orlitzky's site.

The content is edited on Notion and then pulled into a markdown document using a custom built tool based on notion2md. If you want to use this tool too, you can! Head over to the repository at obrhubr/notion-to-jekyll - you can customise it to fit your use-case with the provided command line options.

I use tinylytics for GDPR compliant, privacy respecting analytics.
